What is this?

This project is a task to fullfill NLP college project, which is identifying hate speech from text(mostly from tweet)

The data we use in this repository is from t - davidson with formatted such as :

  • class: 0 - hate speech, 1 - offensive languange, 2 - neither
  • Tweet: tweet/text
